Default ORDER

Secara default beginilah urutannya berdasarkan urutan element di tulis dan juga hierarchy nya.

NODE 1, urutan 1
NODE 1.1, urutan 2
Node 1.1.1, urutan 3
NODE 1.2, urutan 4
NODE 2, urutan 5
NODE 2.1, urutan 6

Di influence oleh position property

Node 1.2 berada di group tersendiri, dibawa ke lapisan teratas dari semua element yang tidak memiliki posisi.

NODE 1, urutan 1
NODE 1.1, urutan 2
Node 1.1.1, urutan 3
NODE 1.2, urutan 4. Saat diberi position relative, urutan nya berubah menjadi teratas.
NODE 2, urutan 5
NODE 2.1, urutan 6

Apabila node lain, misalnya Node 2.1 diberikan posisinya, maka Node ini akan menjadi satu kelompok dengan Node 1.2, tapi yang mana yang paling atas, Kembali di tentukan oleh Node dan hierarchy yang ada pada html.

NODE 1, urutan 1
NODE 1.1, urutan 2
Node 1.1.1, urutan 3
NODE 1.2, urutan POSITION 1
NODE 2, urutan 5
NODE 2.1, urutan POSITION 2 Node ini akan kembali menjadi teratas dan mendahului NODE 1.2. Baik NODE 1.2 dan 2.1 berada di level lain ketimbang seluruh element dengan position static.

Dengan Z-INDEX

Masih menggunakan code HTML & CSS yang paling awal, kali ini NODE 1.2 diberikan z-index: 1, maka node ini akan diberikan group dan layer baru yang bahkan lebih tinggi dan terpisah dari node yang memiliki position biasa, contohnya NODE 2.1.

NODE 1, urutan 1
NODE 1.1, urutan 2
Node 1.1.1, urutan 3
NODE 1.2 Z-index +1
NODE 2, urutan 5
NODE 2.1

Kita juga bisa menciptakan layer paling bawah bahkan lebih dalam dari group element yang tidak diberikan position. Untuk melakukan itu kita akan memberi z-index dengan nilai negatif, contohnya seperti -1.

NODE 1, urutan 1
NODE 1.1, urutan 2
Node 1.1.1, urutan 3
NODE 1.2 Z-index -1
NODE 2, urutan 5
NODE 2.1

Sama halnya dengan element yang diberikan position, apabila ada lebih dari satu element yang berada pada group z-index yang sama, maka element dan hierarchynya yang menentukan urutannya. (Note!: Technically position tanpa z-index adalah position dengan z-index:0)

NODE 1, urutan 1
NODE 1.1, urutan 2
Node 1.1.1, urutan 3
NODE 1.2 Z-index +1
NODE 2, urutan 5
NODE 2.1 Z-index +1
