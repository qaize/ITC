Keunggulan Relative, Child Management

Kita bisa memanfaatkan relative apabila kita ingin merubah posisi-posisi komponent child di dalam parent tanpa ingin mengubah posisi parentnya sama sekali. Kita tidak bisa mencapai ini apabila menggunakan position static, karena position static dari parent tidak terlihat. Kita juga tidak bisa menggunakan position absolute karena posisi parentnya akan tidak dianggap dan parentnya bisa bertumpuk-tumpuk dengan element lain.
