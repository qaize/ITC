Basic Selector
Syntax Name Description CSS Version Specificity
elementname Element Selector Selector ini adalah selector paling sederhana, dimana digunakan untuk memilih langsung seluruh html element/ tag yang akan di style, contohnya seperti div, p, span, dan lain-lain. 1 0,0,1
.class-name Class Selector Selector ini digunakan untuk memilih seluruh element dengan nama class yang ditentukan. 1 0,1,0
#id-name ID Selector Selector ini digunakan untuk memiliki satu element dengan nama id yang ditentukan. Walaupun memungkinkan untuk memilih banyak, tetapi sesuai dengan ketentuan html validation, id tetap bersifat unik. 1 1,0,0
* Select All Selector ini digunakan untuk memilih seluruh element yang ada pada html document. Kurang lebih sifatnya sama seperti pada sql query bahwa * (asterisk) artinya seluruhnya. 2 0,0,0

Complex Selector
Syntax Name Description CSS Version Specificity
elementname1, elementname2 And Selector Selector ini adalah selector yang digunakan untuk memilih dua element atau lebih dengan menggunakan koma sebagai combinatornya. 1 -
containerelement selectedelement Descendant Selector Selector ini adalah selector yang digunakan untuk memilih beberapa element dimana element tersebut merupakan keturunan dari node-node di depannya, ditandai dengan menggunakan spasi sebagai combinatornya. 1 +
parentelement > selectedelement Children Selector Selector ini adalah selector yang digunakan untuk memilih beberapa element dimana element tersebut merupakan anak dari node di depannya, ditandai dengan menggunakan > sebagai combinatornya. 1 +
previouselement + selectelement Next Selector Selector ini adalah selector yang digunakan untuk memilih satu element dimana element tersebut merupakan element yang posisinya tepat setelah previous element. 1 +
previouselement ~ selectelement Preceded Selector Selector ini adalah selector yang digunakan untuk memilih beberapa element dimana element tersebut merupakan element yang posisinya terjadi setelah previous element, baik tepat setelah maupun tidak (masih dalam level yang sama.) 1 +

Attribute Selector
Syntax Name Description CSS Version Specificity
[attribute] Attribute Selector Selector ini digunakan untuk memilih html element dengan men-spesifikasi nama attribute. 2 0,1,0
[attribute=value] / [attribute="value"] Single Value Attribute Selector Selector ini digunakan untuk memilih html element dengan men-spesifikasi isi single value dari attributenya. 2 0,1,0
[attribute~=value] / [attribute~="value"] Alternate Value Attribute Selector Selector ini digunakan untuk memilih html element dengan men-spesifikasi isi salah satu value dari attributenya apabila attribute memiliki lebih dari satu value. 2 0,1,0
[attribute|=value] / [attribute|="value"] Hyphenated Value Attribute Selector Selector ini digunakan untuk memilih html element dimana naming convention dari html attribute tersebut menggunakan - (hyphen) sebagai pemisah satu value attributenya. 2 0,1,0
[attribute^=value] / [attribute^="value"] Initial Value Attribute Selector Selector ini digunakan untuk memilih html element dimana value dari attributenya dimulai dengan kata yang ditentukan. 3 0,1,0
[attribute$=value] / [attribute$="value"] Last Value Attribute Selector Selector ini digunakan untuk memilih html element dimana value dari attributenya diakhiri dengan kata yang ditentukan. 3 0,1,0
[attribute*=value] / [attribute*="value"] Substring Value Attribute Selector Selector ini digunakan untuk memilih html element dimana value dari attributenya mengandung substring yang ditentukan. 3 0,1,0

:out-of-range Out of Range Numeric Digunakan hanya untuk styling macam-macam input element yang memiliki range minimum dan maximum (attribute min dan max), dan kondisi styling memungkinkan saat nilai input berada di luar minimum dan maximum. 3
:read-only Read Only Input Digunakan untuk styling element form input yang memiliki attribute read only. 3
:read-write Editable Input Digunakan untuk styling element form input yang memiliki attribute editable atau inputable. 3

:root Document Root Digunakan untuk seleksi document root, dimana adalah html document tag. 3
:target Target Anchor Digunakan untuk styling target dari anchor link saat dalam keadaan aktif atau dipilih. 3
