Basic HTML
Tag Name Description HTML Semantic
!DOCTYPE Document Type Mendefinisikan tipe document untuk file markup yang ditulis.
html HTML Document Menjelaskan browser bahwa ini adalah html document dan root dari html document.
title Document Title Menentukan judul document pada browser toolbar.
head Document Head Tempat seluruh deklarasi html element yang tidak akan dirender/ditampilkan oleh browser.
body Document Body Tempat seluruh html element yang berisikan page content.
h1 to h6 Headings Digunakan untuk membuat element kepala sebuah content di dalam suatu section.
p Pararaph Digunakan untuk men-define paragraph.
hr Horizontal Rule Digunakan untuk membuat garis horizontal pemisah content di dalam document.

Formatting
Tag Name Description HTML Semantic
abbr Abbreviation Digunakan untuk membuat tooltip yang menjelaskan kepanjangan dari sebuah singkatan
address Address Sebagai tag yang digunakan untuk menunjukan informasi contact dari penulis atau pemilik sebuah article atau document.
b Bold Digunakan untuk membuat text menjadi bold tanpa menggunakan custom styling.
bdo Bi-Directional Override Digunakan untuk menambahkan attribute direction pada text.
blockquote Quotation Digunakan untuk menandakan sebuah section adalah quotation dari sumber lain. (untuk legal formal literature)
cite Citation Digunakan untuk membuat suatu judul (biasanya nama sebuah karya) yang di cite dari pembuatnya. (untuk legal formal literature)
code Code Digunakan untuk membuat formatting satu section dari computer code.
del Del Digunakan untuk membuat text dengan model tercoret, dimana menandakan sebuah koreksi atau indikasi kalau text ini sebenarnya sudah dihapus.
dfn Defining Instance Digunakan untuk menandakan suatu jargon atau kosakata yang ingin dijelaskan definisinya.
em Emphasized Digunakan untuk menandakan suatu penekanan dalam sekumpulan text.
i Italic Digunakan sebagai penanda suatu text dalam alternative voice dan mood.
ins Inserted Digunakan sebagai penanda bahwa text ini adalah text baru sebagai pengganti text yang dihapus.
kbd Keyboard Input Digunakan sebagai penanda bahwa text ini berasal dari input keyboard. Tag ini tidak umum digunakan dan tidak memiliki default style sama sekali.
mark Mark/ Highlighted Digunakan untuk membuat text dalam keadaan kondisi di highlight.
meter Meter Digunakan untuk membuat bar dalam bentuk pengukuran scalar dimana range diketahui. Tag ini belum umum dapat digunakan oleh seluruh browser.
pre Preformatted Text Membuat sekumpulan text yang belum di format ke dalam tampilan html. Atau tampilannya akan persis seperti yang tertulis pada source code.
progress Task Progress Digunakan untuk membuat tampilan dalam progress bar.
q Short Quotation Digunakan untuk membuat semua quotation seseorang dalam tanda petik.
s Strikethrough Pemakaiannya hampir mirip dengan delete tag, hanya saja strikethrough tindak menandaka sesuatu akan digantikan.
samp Sample Output Digunakan untuk menandakan sample output dalam console atau hasil compile dan debugging dari sebuah aplikasi.
small Smaller Text Digunakan untuk membuat text menjadi lebih kecil
strong Strong Menyatakan bahwa sebuah text sangat penting atau diberi penekanan.
sub Subscripted/Subfix Digunakan untuk penulisan sebuah karakter, angka, atau symbol yang formatnya ditulis di bawah normal text-alignment dan biasanya ukuran fontnya pun lebih kecil.
sup Superscript Digunakan untuk penulisan sebuah karakter, angka, atau symbol yang formatnya ditulis di atas normal text-alingnment dan biasanya ukuran fontnya pun lebih kecil.
time Time Digunakan untuk menulis sebuah komponen tanggal dan waktu dari perspective user dan computer.
u Underline Digunakan untuk menggaris bawahi sebuah text di dalam sebuah kalimat.
var Variable Digunakan untuk menulis sebuah variable. Tidak ada specific style dalam tag ini.
wbr Word Break Opportunity Digunakan dalam penulisan sebuah kata yang sangat panjang, apabila ada ukuran browser mengecil, tag ini akan menciptakan kesempatan dan prioritas sebagian kata untuk dibuatkan baris baru.

Programming
